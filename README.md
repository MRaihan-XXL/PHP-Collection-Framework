# PHP Collection Framework

Implementasi framework collection dalam PHP yang terinspirasi dari Java Collection Framework.

### Interface

- `CollectionInterface` - Interface dasar untuk semua koleksi
- `ListInterface` - Interface untuk list (ordered collection)
- `QueueInterface` - Interface untuk queue (FIFO)
- `MapInterface` - Interface untuk map (key-value pairs)
- `IteratorInterface` - Interface untuk iterasi

### Implementasi

- `ArrayList` - Implementasi list menggunakan array
- `LinkedList` - Implementasi list menggunakan linked list
- `Stack` - Implementasi stack (LIFO)
- `Queue` - Implementasi queue (FIFO)
- `HashMap` - Implementasi map menggunakan hashing
